# WebGPUPlayground

This repository serves as a playground for experimenting with and testing the capabilities of WebGPU-native.

## Getting Started

### Prerequisites

- [Xmake](https://xmake.io/#/)

### Building

```bash
xmake
```

### Running

```bash
xmake run
```
